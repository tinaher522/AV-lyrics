#Content of folders

##annotate

Contains the output of the tool */tools/annotation/annotate.py*

##segments

Contains the output of the tool */tools/annotation/segment.py*

##lyrics

##collection

Contains the un processed lyrics files. It is organize by chunks 

###lyrics

Contains the preprocessed lyric file. It is organize by */intrument/chunk*

###json

Contains the jsonified file of each lyric in */lyrics*

##json_lyrics

Contains the union of the lyrics with the annotation. (/segment + /lyrics/json)

##text

Contains the text files of each track. It is organize by */genter/Cover_artist*
